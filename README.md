# AI Munger — 查理芒格投资分析 Agent

以**查理·芒格**投资哲学为核心的 AI 投资分析助手。通过多轮对话，Agent 驱动数据工具和分析技能，生成交互式 HTML 投资分析报告。

## 哲学基础

> "投资并不简单。认为投资简单的人都是傻瓜。" — 查理·芒格

本项目的目标不是替代投资者的判断，而是用芒格的思维框架武装投资者的决策过程——让你像芒格一样思考。

## 功能

- 💬 **自由问答** — "茅台 PE 多少？""搜索一下比亚迪"
- 📊 **深度分析** — 触发词"分析一下""深度研究"，启动 5 阶段分析工作流
- 📄 **HTML 报告** — 含雷达图、趋势图、买入价格区间、可折叠详情面板
- 🇨🇳 **A 股优先** — 腾讯行情 + 东方财富数据，支持与全球龙头对标

## 快速开始

```bash
# 1. 启动 Claude Code
cd D:/HL/ai-munger
claude

# 2. 自由问答
> 茅台现在 PE 多少？

# 3. 深度分析
> 分析一下 600519
```

## 架构

```
👤 用户 → 💬 多轮对话
    ⬇️
🎯 编排层 — munger-orchestrator
    ↙️        ↘️
📡 数据层     🧠 分析层
    ⬇️           ⬇️
🔧 工具层     📊 报告层
```

### 工具层
| 脚本 | 用途 |
|------|------|
| `tools/ashare_data.py` | A 股行情/财务/估值/搜索（零外部依赖） |

### 分析维度

| Skill | 芒格原则 | 分析内容 |
|-------|---------|---------|
| `quality-screen` | "先排除烂公司" | 7 指标量化筛选 |
| `moat-analysis` | "伟大的公司都有护城河" | 5 维度竞争优势评估 |
| `safety-margin` | "以合理价格买伟大公司" | DCF + PE + 清算价值，三档买入区间 |

### 报告示例

生成的报告包含：
- 📋 投资摘要卡片 + 综合评分条
- 🔢 7 指标质量得分卡
- 🏰 5 维度护城河雷达图
- 🛡️ 激进/稳健/保守三档买入价格
- 📈 5 年关键指标趋势图
- ⚠️ 内置免责声明
